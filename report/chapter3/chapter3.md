### User stories Landing Page
| Story ID | Título                        | Descripción                                                                                                                     | Criterios de Aceptación |
|----------|-------------------------------|---------------------------------------------------------------------------------------------------------------------------------|--------------------------|
| US01     | Registro / Get Started        | Como visitante, quiero poder registrarme fácilmente desde el botón “Get Started”, para comenzar a usar la aplicación.           | El sistema permite registrarse desde el botón principal de la landing. |
| US02     | Información de producto       | Como visitante, quiero leer un resumen claro de qué hace el producto, para entender cómo puede ayudarme a gestionar el agua.    | La landing muestra un bloque principal con descripción clara del producto. |
| US03     | Llamada a la acción (CTA)     | Como visitante, quiero ver un botón de “Learn More”, para obtener más detalles antes de registrarme.                            | El sistema despliega información adicional al hacer clic en “Learn More”. |
| US04     | Soluciones por segmento       | Como visitante, quiero conocer las soluciones para negocios y residencias, para identificar cuál se adapta a mi necesidad.      | La landing presenta secciones diferenciadas para cada segmento. |
| US05     | Monitoreo en tiempo real      | Como visitante, quiero saber que el producto ofrece monitoreo en tiempo real, para evaluar si es útil en mi caso.               | La landing incluye texto destacando “Real-Time Monitoring”. |
| US06     | Seguimiento de presión        | Como visitante, quiero saber que el producto mide la presión del agua, para asegurarme de que puede prevenir fallas.            | La landing muestra la función “Pressure Tracking”. |
| US07     | Control de temperatura        | Como visitante, quiero conocer que el sistema mide y controla la temperatura, para evaluar su utilidad en mi entorno.           | La landing muestra la función “Temperature Control”. |
| US08     | Gestión de volumen            | Como visitante, quiero confirmar que el sistema mide el volumen de agua, para evitar desperdicios.                              | La landing muestra la función “Volume Management”. |
| US09     | Ver planes de suscripción     | Como visitante, quiero ver los planes y precios disponibles, para elegir el que mejor se adapte a mi presupuesto.               | La landing muestra secciones de precios con características claras. |
| US10     | Selección de plan             | Como visitante, quiero poder elegir entre los planes Basic, Pro o Enterprise, para suscribirme al que se ajuste a mis necesidades. | La landing muestra botones para seleccionar cada plan. |
| US11     | Preguntas frecuentes (FAQ)    | Como visitante, quiero leer preguntas frecuentes, para aclarar dudas comunes antes de registrarme.                              | La landing muestra un bloque de FAQs expandibles. |
| US12     | Contactar ventas              | Como visitante, quiero tener la opción de contactar al área de ventas, para consultar sobre el plan Enterprise.                 | La landing incluye un botón o formulario de “Contact Sales”. |

### User Stories 

| Story ID | Título                                                                                                                    | Descripción                                                                                                                                      | Criterios de Aceptación                                                        | Relacionado con (Epic ID) |
|----------|---------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------|---------------------------|
| US01     | Monitoreo por habitación                                                                                                  | Como usuario, quiero monitorear en tiempo real el consumo de agua por habitación o dispositivo, para identificar los puntos de mayor consumo.    | El sistema muestra consumo diferenciado por área/dispositivo en el dashboard. | EP01                      |
| US02     | Detección de patrones de uso                                                                                              | Como usuario, quiero identificar patrones de consumo ineficiente (ej. consumos nocturnos altos), para mejorar mis hábitos de consumo.             | El sistema genera insights cuando detecta anomalías en los horarios de consumo. | EP01                      |
| US03     | Riego automático inteligente (dudo la implementacion de esto la vda)                                                      | Como usuario, quiero configurar horarios de riego automático en base a datos de clima y humedad, para evitar desperdicio de agua.                 | El sistema ajusta automáticamente el riego en base a sensores IoT y clima.     | EP01                      |
| US04     | Alertas de fugas (aca si detecta algun consumo excesivo puede avisar de fuga directamente)                                | Como usuario, quiero recibir alertas inmediatas cuando se detecte una fuga en grifos o tuberías, para evitar desperdicios de agua.                | Se envía notificación push/email/SMS en tiempo real al detectar fuga.          | EP02                      |
| US05     | Alertas de exceso de consumo (esto basicamente lo mismo que arriba )                                                      | Como usuario, quiero recibir alertas cuando mi consumo supere un límite definido, para reaccionar a tiempo y reducir gastos.                      | El sistema notifica al superar el umbral configurado por el usuario.           | EP02                      |
| US06     | Alertas de fallas IoT         (aca si no detecta inputs del dispositivo iot que de alerta)                                | Como usuario, quiero recibir notificaciones cuando un dispositivo IoT falle, para solucionarlo rápidamente.                                       | El sistema alerta automáticamente cuando un dispositivo deja de enviar datos.  | EP02                      |
| US07     | Recomendaciones de ahorro    (cuestionable aun )                                                                          | Como usuario, quiero recibir recomendaciones personalizadas según mi consumo, para reducir mi gasto mensual de agua.                             | El sistema genera sugerencias prácticas al menos 1 vez por semana.             | EP03                      |
| US08     | Consejos de sostenibilidad    (consejos basicos lol)                                                                      | Como usuario, quiero recibir consejos prácticos para un uso responsable del agua, para contribuir al cuidado del medio ambiente.                  | El sistema entrega consejos semanales en la app o vía notificación.            | EP03                      |
| US09     | Metas de consumo              (para esto deberiamos esperar primer mes de uso , para entender el consumo del cliente creo) | Como usuario, quiero establecer metas de ahorro de agua, para dar seguimiento a mis progresos y motivarme a reducir consumo.                      | El sistema permite definir un objetivo y muestra indicadores de progreso.      | EP03                      |
| US10     | Historial de facturación      (esto puede ir de la mano con un calculo , dependiendo del precio del agua )                | Como usuario, quiero visualizar mi historial de facturación y anticipar el monto del próximo recibo, para planificar mis gastos.                   | El sistema muestra facturas pasadas y proyecciones de consumo mensual.         | EP04                      |
| US11     | Comparación de consumo        (en el analisis que se muestren meses anteriores y ya )                                     | Como usuario, quiero comparar mi consumo con promedios de mi zona o de hogares similares, para saber si gasto de más.                             | El sistema presenta gráficos comparativos entre consumo propio y promedio.     | EP04                      |
| US12     | Reportes personalizados       (separamos por unidad cada reporte y ya creo cuestionable)                                  | Como institución, quiero recibir reportes detallados y comparativos por área/unidad, para optimizar la gestión del agua.                          | El sistema permite exportar reportes filtrados en PDF/Excel.                   | EP04                      |
| US13     | Inventario de agua en tanques  (un medidor de altura?) no creo que hagamos esta                                           | Como institución, quiero monitorear niveles de agua en tanques, para evitar desabastecimiento.                                                    | El sistema alerta cuando los niveles bajan del umbral definido.                | EP05                      |
| US14     | Predicciones de reabastecimiento  (con respecto al anteior creo que si se pueden implementar ambos )                      | Como institución, quiero recibir predicciones de cuándo se agotará el agua en los tanques, para planificar la reposición a tiempo.                | El sistema estima días restantes de disponibilidad en base al consumo.         | EP05                      |
| US15     | Integración de dispositivos IoT    (si o si)                                                                              | Como usuario, quiero conectar múltiples dispositivos IoT en una sola aplicación, para centralizar la gestión de mi consumo de agua.               | El sistema detecta y sincroniza dispositivos IoT compatibles.                  | EP07                      |
| US16     | Soporte técnico                  (que nos llamen)                                                                         | Como usuario, quiero contar con soporte técnico accesible desde la app, para resolver problemas con mis dispositivos o con la plataforma.         | El sistema ofrece chat, tickets o llamadas de soporte en horario laboral.      | EP07                      |
